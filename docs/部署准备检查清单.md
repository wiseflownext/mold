# 部署前置准备检查清单

## 阿里云ICP备案（最先启动，耗时7-15个工作日）

- [ ] 阿里云账号实名认证
- [ ] 购买ECS（备案需关联云服务器）
- [ ] 提交ICP备案申请（域名+服务器）
- [ ] 等待管局审核
- [ ] 备案通过，配置DNS解析
- [ ] 申请免费SSL证书（DV型）

## 阿里云资源

| 资源 | 规格 | 配置要点 |
|------|------|---------|
| ECS | 2核4G, 40G SSD, Ubuntu 22.04 | 安全组开放80/443/22 |
| RDS MySQL | 1核2G基础版, MySQL 8.0 | 白名单加ECS内网IP |
| OSS | 标准存储, 私有读写 | 配置CORS跨域 |

## DNS子域名规划

| 子域名 | 用途 |
|--------|------|
| api.域名 | 后端API |
| admin.域名 | Web管理后台 |
