# 模具管家 Web管理后台 产品需求文档

## 1. 产品概述

| 项目 | 说明 |
|------|------|
| 产品名称 | 模具管家 Web管理后台 |
| 目标用户 | 管理员、管理层 |
| 登录方式 | 账号 + 密码 |
| 浏览器 | Chrome / Edge |
| 导出格式 | Excel |

---

## 2. 用户角色与权限

| 角色 | 权限范围 |
|------|---------|
| 管理员 | 全部功能：模具录入、验收、鉴定、保养、维修、配置、用户管理 |
| 管理层 | 只读：数据看板、报表、模具查询、告警查看 |

---

## 4. 功能详细说明

### 4.1 数据看板

管理层首页，全部只读。

| 看板模块 | 展示内容 | 图表类型 |
|---------|---------|---------|
| 模具使用率 | 在用数/总数, 闲置率 | 数字卡片 + 环形图 |
| 寿命状态分布 | 各寿命区间(0-50%, 50-80%, 80-90%, 90-100%, 超期)的模具数量 | 柱状图 |
| 保养完成情况 | 本月计划数 / 已完成数 / 逾期数 | 数字卡片 + 进度条 |
| 维修次数/费用 | 近12个月维修次数趋势、累计费用 | 折线图 |
| 当前告警数量 | 寿命预警 / 保养到期 / 使用异常 / 鉴定到期 分类计数 | 数字卡片(红色高亮) |
| 客户模具分布 | 各客户名下模具数量 | 柱状图 |

---

### 4.2 模具台账

**核心页面**，按模具维度管理全部数据。

#### 4.2.1 列表页

**全字段筛选支持：**

| 筛选字段 | 筛选方式 |
|---------|---------|
| 模具编号 | 模糊搜索 |
| 模具类型 | 下拉单选：模压 / 口型 / 接角 |
| 状态 | 下拉多选：在库 / 在用 / 保养中 / 维修中 / 封存 / 待鉴定 / 待验收 / 报废 |
| 所在位置 | 下拉单选：仓库 / 挤出车间 / 模压车间 / 接角车间 / 其他 |
| 客户 | 下拉单选 (从客户列表) |
| 产品名称 | 模糊搜索 |
| 零件号 | 模糊搜索 |
| 首次使用日期 | 日期范围 |
| 寿命使用率 | 范围选择：0-50% / 50-80% / 80-100% / 超期 |
| 设计寿命 | 数值范围 |
| 累计使用量 | 数值范围 |
| 下次保养日期 | 日期范围 |
| 下次鉴定日期 | 日期范围 |

**列表字段：**

模具编号、类型、状态、所在位置、关联客户、关联产品(首个)、设计寿命、累计使用量、使用率、下次保养日期、下次鉴定日期

**操作列：** 查看详情 / 编辑 / 状态变更 / 删除

**批量操作：** 批量导出Excel / 批量删除

#### 4.2.2 详情页

展示模具全部信息，分Tab页：

| Tab | 内容 |
|-----|------|
| 基础信息 | 编号、类型、状态、位置、寿命单位、设计寿命、累计使用量、首次使用日期 |
| 关联产品 | 产品列表(客户、车型、产品名、零件号)，支持增删改 |
| 验收信息 | 验收结果、验收报告文件(可下载)、设计寿命 |
| 使用记录 | 表格：日期、米数、产品、操作人、备注 |
| 领用记录 | 表格：领用/归还、时间、操作人、车间、机台 |
| 保养记录 | 表格：保养日期、保养内容、执行人 |
| 维修记录 | 表格：报修时间、问题描述、照片、处理状态、处理人、费用 |
| 鉴定记录 | 表格：鉴定日期、结果、新周期值、下次鉴定日期、鉴定人/机构、鉴定报告文件 |
| 照片记录 | 时间线展示所有照片 |

#### 4.2.3 模具编辑

**仅管理员可操作。**

可编辑字段：

| 字段 | 编辑规则 |
|------|---------|
| 模具编号 | 不可修改 |
| 模具类型 | 可修改 |
| 所属车间 | 可修改(状态为"在库"时) |
| 寿命单位 | 可修改 |
| 设计寿命 | 可修改(修改后使用率自动重算) |
| 累计使用量 | 可修改(用于数据修正，需填写修改原因) |
| 保养周期 | 可修改 |
| 鉴定周期 | 可修改 |
| 首次使用日期 | 可修改 |
| 关联产品 | 可增删改 |

编辑操作自动记入操作日志(修改前值、修改后值、修改人、修改时间)。

#### 4.2.4 模具删除

**仅管理员可操作。**

| 规则 | 说明 |
|------|------|
| 删除方式 | 软删除(标记删除，数据库保留) |
| 可删除状态 | 仅"在库"和"报废"状态的模具可删除 |
| 不可删除 | 在用/保养中/维修中/待鉴定状态的模具禁止删除 |
| 二次确认 | 删除前弹窗确认，显示模具编号和关联产品数量 |
| 批量删除 | 列表页勾选后批量删除，同样遵循上述状态规则 |
| 恢复 | 管理员可在操作日志中查看已删除模具，支持恢复 |

### 4.3 模具验收

**仅管理员可操作。**

验收是模具进入系统的前置步骤。

#### 验收流程

```
新模具到厂 → 管理员创建验收单 → 填写验收信息 → 通过 → 进入模具录入
                                              → 不通过 → 记录原因，流程结束
```

#### 验收表单字段

| 字段 | 类型 | 必填 |
|------|------|------|
| 模具编号 | 文本输入 | 是 |
| 模具类型 | 下拉选择：模压/口型/接角 | 是 |
| 验收结果 | 单选：通过 / 不通过 | 是 |
| 设计寿命 | 数值输入 + 单位选择(默认:米) | 是 |
| 验收报告 | 文件上传(支持PDF/Word/图片) | 否 |
| 备注 | 文本输入 | 否 |

验收记录列表支持查看历史验收单。

---

### 4.4 模具录入

**仅管理员可操作。前提：验收通过。**

#### 录入表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 模具编号 | 从验收通过的记录中选择 / 自动带入 | 是 | |
| 模具类型 | 自动带入 | 是 | |
| 所属车间 | 下拉选择 | 否 | 初始可为空(仓库) |
| 设计寿命 | 自动带入(验收时填的) | 是 | |
| 寿命单位 | 下拉选择，默认"米" | 是 | 米/模次/自定义 |
| 保养周期 | 数值 + 单位(天/米) | 否 | |
| 鉴定周期 | 数值 + 单位(天/米) | 否 | |
| 首次使用日期 | 日期选择 | 否 | |
| 关联产品 | 子表单(可多条)：客户、车型、产品名、零件号 | 否 | |

录入成功后模具状态为"在库"，位置为"仓库"。

#### 批量导入

支持Excel/JSON格式批量导入(兼容现有3个JSON文件格式)。

导入时需指定模具类型，导入的模具直接进入"在库"状态(跳过验收流程，用于初始化历史数据)。

---

### 4.5 模具状态变更

管理员可手动变更模具状态：

| 操作 | 起始状态 | 目标状态 | 需要填写 |
|------|---------|---------|---------|
| 封存 | 在库 | 封存 | 封存原因 |
| 解封 | 封存 | 在库 | - |
| 报废 | 在库/封存/维修中 | 报废 | 报废原因 |
| 送保养 | 在库/在用 | 保养中 | 保养内容 |
| 保养完成 | 保养中 | 在库 | 保养结果 |
| 送维修 | 在库/在用 | 维修中 | 故障描述 |
| 维修完成 | 维修中 | 在库 | 维修结果、费用 |
| 送鉴定 | 在库 | 待鉴定 | - |
| 鉴定完成 | 待鉴定 | 在库 | 鉴定结果(见4.7) |

所有状态变更记录系统自动留痕(操作人、时间、变更内容)。

### 4.6 寿命监控

| 功能 | 说明 |
|------|------|
| 寿命列表 | 表格展示所有模具的设计寿命、当前累计、使用率，按使用率降序排列 |
| 预警筛选 | 快捷筛选：80-90% / 90-100% / 超期 |

### 4.7 周期鉴定

**仅管理员可操作。**

触发场景：定期强制鉴定 + 出现质量问题时触发。

#### 鉴定记录表单

| 字段 | 类型 | 必填 |
|------|------|------|
| 关联模具 | 选择 | 是 |
| 鉴定日期 | 日期 | 是 |
| 鉴定结果 | 单选：合格 / 不合格 / 调整周期 | 是 |
| 新周期值 | 数值(鉴定结果为"调整周期"时显示) | 条件必填 |
| 下次鉴定日期 | 日期 | 是 |
| 鉴定人/机构 | 文本 | 是 |
| 鉴定报告 | 文件上传 | 否 |
| 备注 | 文本 | 否 |

**系统行为：**
- 结果为"调整周期"时，自动更新模具的设计寿命为新周期值
- 自动更新模具的"下次鉴定日期"
- 结果为"不合格"时，模具状态强制变为"封存"

---

### 4.8 保养管理

#### 4.8.1 保养计划

- 每个模具可配置保养周期(天数 或 使用米数)
- 系统根据周期自动计算下次保养日期
- 保养计划列表：展示所有模具的下次保养时间，支持按紧急度排序
- 逾期保养标红

#### 4.8.2 保养记录

| 字段 | 类型 | 必填 |
|------|------|------|
| 关联模具 | 选择 | 是 |
| 保养日期 | 日期 | 是 |
| 保养内容 | 文本 | 是 |
| 执行人 | 选择 | 是 |
| 备注 | 文本 | 否 |

保养完成后自动重新计算下次保养日期。

| **周期鉴定** | 创建/查看鉴定记录 | O | X | X |

| **操作日志** | 查看 | O | X | O |

O = 有权限，X = 无权限

**角色管理功能：**

| 功能 | 说明 |
|------|------|
| 角色列表 | 展示所有角色(预设+自定义) |
| 新建角色 | 输入角色名，勾选权限项 |
| 编辑角色 | 修改权限配置(预设角色不可删除，但可调整权限) |
| 删除角色 | 仅自定义角色可删除，删除前需确认无用户使用该角色 |

**权限配置界面：**
- 左侧：菜单树(勾选可访问的菜单)
- 右侧：每个菜单下的操作权限(查看/新增/编辑/删除/导出)

字段：操作人、操作时间、操作类型、操作内容、IP地址。

## 5. 模具完整状态流转图

```
                    ┌──────────┐
        新模具到厂 → │  待验收   │
                    └────┬─────┘
                验收通过 │  验收不通过 → 记录原因,结束
                    ┌────▼─────┐
          录入系统 → │   在库    │ ←──── 归还 / 保养完成 / 维修完成 / 解封 / 鉴定完成(合格)
                    └────┬─────┘
                         │
            ┌────────────┼────────────┬──────────────┬──────────────┐
            ▼            ▼            ▼              ▼              ▼
       ┌────────┐  ┌─────────┐  ┌─────────┐   ┌─────────┐   ┌─────────┐
       │  在用   │  │  保养中  │  │  维修中  │   │  封存   │   │  待鉴定  │
       └───┬────┘  └─────────┘  └────┬────┘   └────┬────┘   └─────────┘
           │                         │              │
           └─────────────────────────┴──────────────┘
                                     │
                                     ▼
                               ┌──────────┐
                               │   报废    │ (管理员直接操作,终态)
                               └──────────┘
```

合法状态变更路径：

| 当前状态 | 可变更为 |
|---------|---------|
| 待验收 | → 在库(验收通过录入) |
| 在库 | → 在用 / 保养中 / 维修中 / 封存 / 待鉴定 / 报废 |
| 在用 | → 在库(归还) / 保养中 / 维修中 |
| 保养中 | → 在库(保养完成) |
| 维修中 | → 在库(维修完成) / 报废 |
| 封存 | → 在库(解封) / 报废 |
| 待鉴定 | → 在库(合格) / 封存(不合格) |
| 报废 | 终态，不可变更 |
