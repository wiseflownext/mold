# 模具管家 APP端 产品需求文档

## 1. 产品概述

| 项目 | 说明 |
|------|------|
| 产品名称 | 模具管家 APP |
| 技术方案 | Flutter (iOS + Android) |
| 目标用户 | 工厂车间操作工、管理员、管理层 |
| 登录方式 | 账号 + 密码 |

---

## 2. 双模式设计

APP根据登录角色呈现**两套完全不同的界面**，登录后自动切换。

| | 工人模式 (操作工) | 管理模式 (管理员/管理层) |
|--|---|---|
| 核心定位 | **快速操作 + 提醒驱动** | **数据掌控 + 决策辅助** |
| 设计原则 | 大按钮、少输入、戴手套可操作 | 信息密度高、图表可视化 |
| 首页 | 待办提醒 + 快捷操作按钮 | 数据看板(图表+指标) |
| 底部Tab | 首页 / 模具 / 消息 / 我的 | 看板 / 模具 / 告警 / 我的 |
| 模具列表 | 简化卡片(编号+产品+状态) | 完整信息(含寿命、保养、鉴定日期) |
| 模具详情 | 底部有操作按钮(记录/报修/拍照) | 无操作按钮，侧重数据查看 |

角色和权限由Web后台统一配置，支持自定义角色并指定走哪种APP模式。

---

## 3. 工人模式

面向车间操作工。**极简操作 + 提醒驱动。**

### 3.1 首页（工人版）

**底部Tab：** 首页 / 模具 / 消息 / 我的

**布局（从上到下）：**

| 区域 | 内容 | 说明 |
|------|------|------|
| 欢迎栏 | "你好，张三" + 角色标签 | 顶部 |
| 待办提醒 | 红色卡片：待归还模具数、待确认工单数 | 有待办时显示，无待办时隐藏 |
| 告警横幅 | 滚动显示最新告警 | 如"5700-SY118-59 寿命达90%" |
| 快捷操作 | 4个大按钮(2x2): 记录使用 / 领用归还 / 报修 / 拍照 | 核心区域，按钮高度80px |
| 最近使用 | 最近操作过的3个模具卡片(横滑) | 点击直接进详情 |

### 3.2 模具查询（工人版）

**入口：** 底部Tab "模具"

#### 搜索
- 搜索框：模具编号 / 产品名 / 客户名 / 零件号，模糊匹配
- 最近查看：展示最近5个，减少重复搜索

#### 筛选

| 筛选项 | 类型 |
|--------|------|
| 模具类型 | 单选：模压 / 口型 / 接角 / 全部 |
| 状态 | 多选：在库 / 在用 / 保养中 / 维修中 / 封存 / 待鉴定 |
| 所在车间 | 单选：挤出车间 / 模压车间 / 接角车间 / 仓库 / 全部 |
| 客户 | 单选 |

#### 列表卡片

每条展示：模具编号(大字) + 产品名 + 客户名 + 状态标签 + 寿命进度条

### 3.3 模具详情（工人版）

| 信息区域 | 展示内容 |
|---------|---------|
| 基础信息 | 模具编号、类型、当前状态、所在位置 |
| 寿命信息 | 进度环(当前/设计寿命) |
| 关联产品 | 客户、车型、产品名、零件号 |
| Tab页 | 使用记录 / 保养记录 / 维修记录 |

**底部固定操作栏（工人版独有）：**
- 记录使用 (主色按钮)
- 报修 (红色轮廓按钮)
- 拍照 (灰色轮廓按钮)

### 3.4 记录使用

**操作流程：**
1. 选择模具 (搜索 / 从"最近使用"快捷选取)
2. 填写本次生产米数 (数字键盘)
3. 选择生产产品 (从该模具关联的产品列表中选)
4. 操作人 (自动填当前登录人)
5. 日期 (默认今天，可修改)
6. 备注 (选填)
7. 提交

**系统行为：**
- 提交后累计使用量自动叠加
- 若触发寿命预警阈值，自动生成告警

### 3.5 领用 / 归还

**页面顶部Tab切换：** 领用 | 归还

#### 领用
1. 选择模具 (仅显示"在库"状态)
2. 选择目标车间
3. 选择机台 (选填)
4. 确认领用

**系统行为：** 状态"在库"→"在用"，记录领用人、时间、去向

#### 归还
1. 选择模具 (仅显示"在用"状态，"我领用的模具"快捷选取)
2. 确认归还

**系统行为：** 状态"在用"→"在库"，回到仓库

### 3.6 报修提报

1. 选择模具
2. 拍照 (至少1张，最多9张)
3. 问题描述
4. 提交

**系统行为：** 生成维修工单(待处理) + 通知管理员

### 3.7 拍照记录

1. 拍照 / 从相册选取
2. 文字备注 (选填)
3. 提交

**系统行为：** 照片关联模具，自动记录拍摄人和时间

### 3.8 消息中心（工人版）

**入口：** 底部Tab "消息"

| 消息类型 | 触发条件 |
|---------|---------|
| 维修工单更新 | 自己提报的工单状态变更 |
| 保养提醒 | 自己领用的模具保养到期 |
| 归还提醒 | 领用超时未归还 |

仅显示与自己相关的消息，未读/已读，支持全部已读。

### 3.9 个人中心（工人版）

| 功能 | 说明 |
|------|------|
| 个人信息 | 姓名、角色、所属车间 (只读) |
| 我的领用 | 当前未归还模具清单 (含数量角标) |
| 修改密码 | 旧密码 + 新密码 |
| 退出登录 | - |

## 4. 管理模式

面向管理员和管理层。**数据掌控 + 决策辅助。**

### 4.1 首页/数据看板（管理版）

**底部Tab：** 看板 / 模具 / 告警 / 我的

**布局（从上到下）：**

| 区域 | 内容 | 说明 |
|------|------|------|
| 告警卡片 | 当前告警总数，红色高亮 | 点击进入告警列表 |
| 关键指标(横滑4卡片) | 模具总数 / 在用率 / 本月维修次数 / 保养完成率 | 大数字 + 趋势箭头 |
| 寿命预警列表 | 使用率>80%的模具，按使用率降序 | 最多10条，点击进详情 |
| 保养到期列表 | 即将到期 + 已逾期的模具 | 逾期标红 |
| 客户分布图 | 各客户模具占比 | 饼图/环形图 |
| 维修趋势图 | 近6个月维修次数 | 折线图 |

### 4.2 模具列表（管理版）

与工人版共用列表页，但信息更丰富：

| 差异点 | 工人版 | 管理版 |
|--------|--------|--------|
| 卡片信息 | 编号+产品+客户+状态+进度条 | 额外显示：下次保养日期、下次鉴定日期 |
| 筛选项 | 类型/状态/车间/客户 | 额外增加：寿命区间(0-80%/80-100%/超期)、保养状态(正常/即将到期/逾期) |
| 排序 | 按编号 | 可切换：按编号 / 按使用率 / 按保养紧急度 |

### 4.3 模具详情（管理版）

与工人版共用详情页，但差异如下：

| 差异点 | 工人版 | 管理版 |
|--------|--------|--------|
| 底部操作栏 | 记录使用/报修/拍照 | **无操作栏** |
| 寿命信息 | 进度环 | 进度环 + 预计到期时间 + 日均消耗速率 |
| Tab页 | 使用记录/保养/维修 | 额外增加：鉴定记录、照片记录 |

### 4.4 告警中心（管理版 - 独立Tab）

管理模式底部Tab第3个是"告警"，侧重告警管理而非普通消息。

| 区域 | 内容 |
|------|------|
| 告警统计(顶部) | 4个数字：寿命预警 / 保养到期 / 使用异常 / 鉴定到期 |
| 告警列表 | 默认显示未处理，按紧急度排序 |
| 筛选Tab | 全部 / 未处理 / 已处理 |
| 快捷操作 | 滑动卡片可"标记已处理" |

**告警类型：**

| 告警类型 | 触发条件 |
|---------|---------|
| 寿命预警 | 累计使用量达到设计寿命的80% / 90% |
| 保养到期 | 距下次保养日期 ≤ N天 (可配置) |
| 使用异常 | 单次使用量超正常值N倍 (可配置) |
| 鉴定到期 | 距下次鉴定日期 ≤ N天 (可配置) |

### 4.5 个人中心（管理版）

| 功能 | 说明 |
|------|------|
| 个人信息 | 姓名、角色 (只读) |
| 修改密码 | 旧密码 + 新密码 |
| 退出登录 | - |

## 6. 交互设计原则

### 6.1 工人模式

| 原则 | 要求 |
|------|------|
| 大按钮 | 主操作按钮高度 ≥ 48px，便于戴手套点击 |
| 少打字 | 能选择的不让输入，数字用数字键盘 |
| 快捷选取 | 最近使用/最近查看，减少重复搜索 |
| 操作反馈 | 每次提交后有明确的成功/失败提示 |
| 离线提示 | 无网络时明确提示，不允许提交 |

### 6.2 管理模式

| 原则 | 要求 |
|------|------|
| 信息密度 | 卡片可展示更多字段，表格紧凑 |
| 图表优先 | 关键数据用图表可视化(饼图/折线/进度条) |
| 下钻能力 | 看板数字可点击，跳转到对应筛选后的列表 |
| 快速处理 | 告警列表支持滑动快捷处理 |

### 6.3 共享规范

| 项 | 值 |
|------|------|
| 状态颜色 | 在库=#52C41A 在用=#1677FF 保养=#FA8C16 维修=#F5222D 封存=#8C8C8C 待鉴定=#FAAD14 待验收=#722ED1 报废=#434343 |
| 寿命进度条 | 0-50%绿 50-80%蓝 80-90%黄 90-100%橙 超期红 |
| 状态标签 | 圆角矩形，背景为状态色10%透明度，文字为状态色 |

## 7. 寿命单位说明

| 配置项 | 说明 |
|--------|------|
| 默认单位 | 米 (适用于挤出类模具) |
| 可选单位 | 米 / 模次 / 其他自定义单位 |
| 配置级别 | 每个模具独立配置 |
| 显示格式 | "12.5万米 / 20万米 (62.5%)" |
