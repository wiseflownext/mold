# 模具管家 Web管理后台 原型设计文档

> 本文档用于 Figma Make 生成 Web 管理后台原型。请严格按照以下规范生成。
> 本产品有配套的移动端 APP，两端共享同一套设计规范（状态颜色、数据模型）。

---

## 设计规范

### 产品信息
- 产品名称：模具管家 - 管理后台
- 平台：Web (桌面端浏览器)
- 设计尺寸：1440 x 900
- 设计风格：企业级后台管理系统，左侧导航 + 顶部面包屑 + 右侧内容区
- 语言：中文

### 色彩体系

**主色：**
- 主色 Primary: #1677FF
- 主色深 Primary Dark: #0958D9

**状态色（与APP端统一）：**
- 在库: #52C41A (绿色)
- 在用: #1677FF (蓝色)
- 保养中: #FA8C16 (橙色)
- 维修中: #F5222D (红色)
- 封存: #8C8C8C (灰色)
- 待鉴定: #FAAD14 (黄色)
- 待验收: #722ED1 (紫色)
- 报废: #434343 (深灰)

**功能色：**
- 成功: #52C41A
- 警告: #FAAD14
- 错误: #F5222D
- 信息: #1677FF

**中性色：**
- 标题文字: #1F1F1F
- 正文文字: #434343
- 辅助文字: #8C8C8C
- 表头背景: #FAFAFA
- 分割线: #F0F0F0
- 页面背景: #F5F5F5
- 卡片/表格背景: #FFFFFF

### 字体规范
- 页面标题: 20px Bold
- 卡片标题: 16px Medium
- 表格正文: 14px Regular
- 辅助信息: 12px Regular
- 看板数字: 32px Bold

### 布局规范
- 左侧导航宽度: 220px (可收起为60px)
- 顶部栏高度: 56px
- 内容区内边距: 24px
- 卡片圆角: 8px
- 按钮圆角: 6px
- 表格行高: 48px

### 寿命进度条颜色规则（与APP统一）
- 0-50%: #52C41A (绿)
- 50-80%: #1677FF (蓝)
- 80-90%: #FAAD14 (黄)
- 90-100%: #FA8C16 (橙)
- 超过100%: #F5222D (红)

### 状态标签样式
- 形式：圆角矩形标签 (圆角4px, 内边距2px 8px)
- 背景色：对应状态色的10%透明度
- 文字色：对应状态色
- 字号：12px
